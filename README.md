# Suraksha AI

Real-time AI-powered document fraud detection system for banking and underwriting workflows.

## Features

- OCR extraction
- NLP document classification
- Metadata forensic analysis
- CV fraud detection
- Font alignment analysis
- Risk scoring engine
- React forensic dashboard

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
